# AI Tools Hub

A futuristic Web3 dashboard to discover, organize, and access your favorite AI tools for daily productivity.

## Features

- Modern, glassmorphic UI with animated particle background
- Curated cards for top AI tools (ChatGPT, Gemini, Grok, Claude, DeepSeek, Vercel v0, and more)
- Responsive design, beautiful gradients, and smooth interactions
- Quick launch buttons for each tool

## Tech Stack

- Next.js (App Router)
- React
- Tailwind CSS

## Live Demo

[https://mydailyaidrivers.onrender.com/](https://mydailyaidrivers.onrender.com/)

## Getting Started

1. Clone the repo
2. Install dependencies: `npm install`
3. Run locally: `npm run dev`
